• Why Java restricts a file to one public class•

Java me rule hai:

👉 Ek .java file me sirf ek hi public class ho sakti hai
👉 Aur file ka naam us public class ke naam jaisa hi hona chahiye

Example:

public class Student {
    
}

File name:

Student.java

Agar aap likho:

public class Student {

}

public class Teacher {

}

❌ Error aayega

Kyuki Java allow nahi karta multiple public classes in one file.

2️⃣ Java Compiler ka Logic

Java compiler (javac) file ko compile karta hai.

Example:

Student.java

Usme agar ho:


public class Student {

}

To compiler Student.class file banata hai.

Agar 2 public class ho:

Student
Teacher

To compiler confuse ho jayega:

❓ File ka naam kya hona chahiye?

Student.java
Teacher.java

Isliye Java rule bana diya:

✅ One file = one public class

1️⃣ Sabse Pehle – Wrapper Class Kya Hoti Hai
********************************************************************************

Java me primitive data types hote hain:

byte
short
int
long
float
double
char
boolean

Ye primitive values store karte hain.

Lekin kabhi-kabhi Java ko object chahiye hota hai.

Is problem ko solve karne ke liye Java ne Wrapper Classes banayi.

👉 Wrapper Class = Primitive data type ko object me convert karne wali class

Example:

int  → Integer
double → Double
char → Character
boolean → Boolean

Example code:

int num = 10;

Integer obj = Integer.valueOf(num);

Yaha:

num → primitive
obj → object

3️⃣ Wrapper Classes Kyun Exist Karti Hain
********************************************************************************

Primitive types object nahi hote.

Lekin Java me bahut jagah object required hota hai.

Example:

❌ Primitive allowed nahi

ArrayList<int>

Java error dega.

✅ Wrapper class allowed

ArrayList<Integer>

Example:

import java.util.*;

ArrayList<Integer> list = new ArrayList<>();

list.add(10);
list.add(20);

Isliye wrapper classes exist karti hain.

4️⃣ Primitive vs Object Difference

| Primitive             | Wrapper Object         |
| --------------------- | ---------------------- |
| value store karta hai | object store karta hai |
| fast hota hai         | thoda slow             |
| methods nahi hote     | methods available      |
| memory kam use        | memory zyada use       |


Primitive:

int a = 10;

Wrapper:

Integer a = 10;

6️⃣ Autoboxing
***************************************************************

Java 5 ke baad autoboxing introduce hua.

👉 Primitive automatically object me convert ho jata hai.

Example:

int a = 10;

Integer obj = a;

Behind the scene:

Integer obj = Integer.valueOf(a);

Example:

ArrayList<Integer> list = new ArrayList<>();

list.add(10);

Behind the scene:

list.add(Integer.valueOf(10));

7️⃣ Unboxing
************************************************

Object ko wapas primitive me convert karna.

Example:

Integer obj = 50;

int num = obj;

Behind the scene:

int num = obj.intValue();

Example:

Integer a = 100;

int b = a;

✅ Simple Definition

1️⃣ POJO Class Kya Hoti Hai
****************************************************************************************************

POJO = Plain Old Java Object

Simple language me:

👉 Ek simple Java class jisme koi special restriction ya framework dependency na ho.

Matlab:

normal variables

getter setter

constructor

methods

Bas ek simple Java object.

Example:

public class Student {

    private int id;
    private String name;

    public Student() {
    }

    public Student(int id, String name) {
        this.id = id;
        this.name = name;
    }

    public int getId() {
        return id;
    }

    public void setId(int id) {
        this.id = id;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }
}

Ye ek perfect POJO class hai.

2️⃣ POJO Class Ki Characteristics

Ek class ko POJO hone ke liye usually ye features hote hain:

1️⃣ Private variables (encapsulation)
2️⃣ Public getters and setters
3️⃣ Default constructor
4️⃣ Koi framework dependency nahi
5️⃣ Simple Java logic
