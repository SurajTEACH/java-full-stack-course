📄 Food Stall Menu Driven System (Java) – Documentation
1. Project Overview

Project Name: Food Stall Menu Driven System
Technology: Java
Concepts Used:

Data Types

Arrays

If-Else

Functions

Menu Driven Logic

Purpose:
Food Stall पर आने वाले customer को menu दिखाना, item select करवाना, quantity लेना और total bill calculate करना।

2. System Flow (Working Process)

System का flow कुछ इस प्रकार होगा:

Program start होगा

User को food menu दिखेगा

User item select करेगा

Quantity डालेगा

Bill calculate होगा

Final total amount दिखेगा

Exit option मिलेगा

Flow Diagram (Logic):

Start
  |
Show Menu
  |
User Choice
  |
Check Item (If-Else)
  |
Enter Quantity
  |
Calculate Price
  |
Add to Total Bill
  |
Show Total
  |
Exit
3. Data Structures Used
1️⃣ Arrays

int prices[]

Example Data:

4. Functions Design

Industry style में पहले functions decide किए जाते हैं।

Function Name	Purpose
showMenu()	Food menu display करना
getChoice()	User से item choice लेना
getQuantity()	Quantity लेना
calculateBill()	Price calculate करना
displayBill()	Final bill show करना
main()	Program control
5. Function Details
1️⃣ showMenu()
Purpose

User को food stall का menu दिखाना।

Input

No input

Output

Menu display

Logic
Print Food Stall Menu
Show all food items
Show price of each item
Show Exit option

6️⃣ main()
Purpose

Program execution control।

Logic
Start program

Call showMenu()

Take user choice

If choice is valid
     Take quantity
     Calculate bill
     Display bill

Else if choice == Exit
     Stop program

Else
     Show invalid choice
6. Menu Driven Logic

Menu driven system का मतलब:

User बार-बार menu देख सकता है और item select कर सकता है।

Example logic:

if(choice == 1)
Burger

else if(choice == 2)
Pizza

else if(choice == 3)
Sandwich

else if(choice == 4)
Momos

else if(choice == 5)
ColdDrink

else if(choice == 6)
Exit

else
Invalid choice
